# Парсер вакансий с hh.ru

## Описание
Программа получает данные о 10 компаниях и их вакансиях с сайта hh.ru, сохраняет в PostgreSQL и предоставляет аналитику.

## Установка
1. Установи PostgreSQL
2. Создай базу данных hh_project
3. Установи зависимости: `pip install -r requirements.txt`
4. Настрой .env файл

## Запуск
```bash
python main.py
